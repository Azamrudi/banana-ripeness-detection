# 🍌 Banana Ripeness Detection (Manual Method - No Machine Learning)

## 📌 Overview
Project ini bertujuan untuk mendeteksi tingkat kematangan buah pisang menggunakan metode pengolahan citra digital berbasis warna (HSV) tanpa menggunakan machine learning.

## 🎯 Objective
Mengklasifikasikan pisang menjadi:
- Mentah (Hijau)
- Matang (Kuning)
- Busuk (Coklat)

## 🧠 Method
Metode yang digunakan:
1. Image preprocessing
2. Konversi ke HSV
3. Color segmentation
4. Pixel counting
5. Rule-based classification

## ⚙️ Technologies
- Python
- OpenCV
- NumPy
- Matplotlib
